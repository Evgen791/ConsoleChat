	Данный код представляет собой консольное приложение чата, которое позволяет пользователям зарегистрироваться, войти в систему, отправлять сообщения, выходить и просматривать список пользователей и сообщений в чате.

	В функции main происходит обработка пользовательского ввода и вызываются функции для выполнения соответствующих действий в чате.
Функция print_users выводит список пользователей, подключенных к чату, а функция print_messages - список сообщений, которые были отправлены в чат.
Чтобы зарегистрироваться в чате, пользователь должен выбрать команду "register" и ввести свое имя пользователя и пароль. Чтобы войти в систему, пользователь должен выбрать команду "login" и ввести свои учетные данные.
Для отправки сообщения пользователь должен сначала войти в систему, а затем выбрать команду "send" и ввести текст сообщения.
Команда "logout" позволяет пользователю выйти из системы, а команда "quit" - завершить работу приложения.
В случае ввода неверной команды выводится сообщение "Invalid command".

	Реализован класс Chat, отвечающий за логику работы чата.
Он предоставляет методы для регистрации пользователей, входа/выхода из системы, отправки сообщений и получения списка пользователей и сообщений.
 
Класс также хранит информацию о пользователях (в виде карты) и о текущем авторизованном пользователе (в виде указателя).


	Реализован класс Message для хранения информации о сообщениях в чате.
В конструкторе класса задаются отправитель и текст сообщения, а также устанавливается метка времени на текущую дату и время, используя функцию std::time(nullptr).
Для доступа к данным сообщения используются методы get_sender(), get_text() и get_timestamp(), которые возвращают соответствующие поля класса.
Класс Message определяет свойства и поведение объектов сообщений, позволяя программе легко обрабатывать и управлять сообщениями в чате.
	
	Клас User хранит информацию о пользователях в чате.
Имеет два приватных поля: имя пользователя и пароль, а также свойство online, которое указывает, находится ли пользователь в данный момент в сети.
Конструктор класса позволяет задать имя пользователя и пароль при создании объекта.
Для доступа к данным пользователя используются методы get_name() и verify_password(), которые возвращают имя пользователя и проверяют, соответствует ли введенный пароль паролю пользователя.
Для управления статусом пользователя (в сети или нет) предоставлены методы is_online() и set_online(), которые позволяют устанавливать и получать соответствующее свойство объекта.
Класс User позволяет программе легко обрабатывать и управлять пользователями в чате.
